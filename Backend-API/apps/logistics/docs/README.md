# Módulo de logística

Este app centraliza transportistas, envíos y cuentas corrientes.

Áreas cubiertas:
- Maestro de transportistas (datos comerciales, bancarios, condiciones).
- Gestión de envíos para compras y ventas con tracking y costos.
- Cuenta corriente de transportistas para imputar fletes y pagos.

Próximos pasos:
1. Definir serializers/viewsets para transportistas y envíos.
2. Integrar con órdenes de compra y pedidos para crear envíos automáticamente.
3. Conectar con Tesorería/Payables para liquidar las cuentas corrientes.
